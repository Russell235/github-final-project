# Simple Interest Calculator

A simple and beginner-friendly project that calculates **Simple Interest** using basic financial formulas.  
This project is useful for students, beginners in programming, and anyone learning financial mathematics.

---

## 📌 Formula Used

The formula for Simple Interest is:

\[
SI = \frac{P \times R \times T}{100}
\]

Where:

- **P** = Principal amount
- **R** = Rate of interest
- **T** = Time period
- **SI** = Simple Interest
